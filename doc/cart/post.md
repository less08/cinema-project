# Add concert session to shopping cart

Adding the concert session to shopping cart

**URL** : `/shopping-carts

**Method** : `POST`

**Auth required** : YES

**Allowed for roles** : USER

**Query params**:   

    concertSessionId: number - required

**Query params example**:

    concertSessionId=1
```

## Success Responses
**Code** : `200 OK`

{}
